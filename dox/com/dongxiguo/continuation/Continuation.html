Class Fields

static function cpsByMeta(metaName:Dynamic):Dynamic

A build macro that enables CPS transformation for the annotated class.

In @:build(com.dongxiguo.continuation.Continuation.cpsByMeta("youMeta")) class YourClass, all @youMeta methods in YourClass will be transfromed to CPS functions. In these @youMeta methods, some macros are performed to enable the magic .async() suffix syntax that invokes other asynchronous functions.

static function cpsFunction(expr:Dynamic):Dynamic
